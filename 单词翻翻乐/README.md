# 🎴 单词翻翻乐

> 中英文单词记忆匹配游戏，在游戏中巩固词汇量！

## 🎮 玩法

- 牌面朝下，点击翻开两张牌
- 若两张牌的单词意思相同则配对成功
- 全部配对完成即通关
- 支持**自定义词库**导入

## ✨ 特点

- 多种内置词库（动物、水果、电子设备等）
- 支持导入自定义词库（JSON/YAML/TXT）
- 翻转动画流畅
- 记录用时与步数

## 📁 词库格式

```json
{
  "words": [
    { "en": "apple", "zh": "苹果" },
    { "en": "banana", "zh": "香蕉" }
  ]
}
```

## 🛠️ 技术

HTML + CSS + JavaScript，词库文件在 `单词翻翻乐词库/` 目录下。
