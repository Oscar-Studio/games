# ♔ 国际象棋

经典国际象棋对弈游戏，支持双人同屏对战。

## 功能

- ♔♕♖♗♘♙ 完整的棋子 Unicode 显示
- 🚩 王车易位（短移位 / 长移位）
- 🏃 吃过路兵
- ♟️ 兵升变（可选择升变为后/车/象/马）
- ⚠️ 将军提示
- 🏆 将死检测
- 📊 吃子记录

## 规则

- 白方先手，对弈双方轮流移动
- 吃过路兵、升变等特殊规则自动处理
- 点击己方棋子显示可移动位置，再点击目标位置移动
- 点击其他地方取消选择

## 操作说明

1. 点击己方棋子选中（高亮显示）
2. 可移动位置会显示圆点提示，可吃掉的位置显示圆圈
3. 点击目标位置完成移动
4. 点击空白或其他己方棋子切换选中

## 技术

- 纯 HTML5 + CSS3 + JavaScript，无外部依赖
- 使用 Unicode 棋子字符，无需加载图片
- 响应式布局，适配不同屏幕
