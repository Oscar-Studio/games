# 🧊 3D 魔方

> 在浏览器中还原经典三阶魔方，支持滑动转面、视角旋转、打乱/撤销/计时。

## 🎮 玩法

- **滑动魔方** — 在魔方上朝任意方向滑动，对应面会旋转 90°
- **拖动空白处** — 旋转视角，从不同角度观察魔方
- **滚轮 / 双指捏合** — 缩放视角
- **打乱** — 一键随机打乱（25 步），开始计时
- **撤销** — 撤销上一步操作
- **还原** — 恢复到初始已还原状态

## ✨ 特点

- **纯 3D 渲染** — Three.js 实现 27 个 cubie 子方块，每个贴纸独立着色
- **物理正确** — 旋转轴/方向遵循右手法则，face 旋转后 cubie 的位置和朝向严格同步
- **完整功能** — 打乱、撤销、计时、步数统计、自动胜利检测
- **跨平台** — 鼠标 + 触屏 + 键盘全部支持
- **响应式** — 桌面与移动端统一体验

## 🕹 操作

| 操作 | 桌面端 | 移动端 |
|---|---|---|
| 转面 | 在魔方上滑动 | 在魔方上滑动 |
| 旋转视角 | 拖动空白处 | 拖动空白处 |
| 缩放 | 鼠标滚轮 | 双指捏合 |
| 转面快捷键 | `F` `B` `L` `R` `U` `D`（+ `Shift` 反向） | — |
| 打乱 / 撤销 / 还原 | 按钮 | 按钮 |

## 🛠 技术

- **Three.js 0.160**（CDN 引入）— 3D 渲染
- **纯 HTML + CSS + 原生 JavaScript**，无构建步骤
- **Cubie 模型** — 27 个子方块，每个记录逻辑位置 (`gridPos`) 和累积朝向 (`quaternion`)
- **临时 Group 旋转** — 转面时把 9 个 cubie 临时挂到一个 Group 上旋转，动画结束回写到主 Group，保证视觉与逻辑严格同步
- **Raycaster 拾取** — pointerdown 时判断点击落在魔方上还是空白处，严格互斥地分发到「转面模式」或「视角模式」
